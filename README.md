# Java Annotation Repository

 ## 📚 Learning Goals
  In this repository the exercise is create a annotation with value that contains a table name.
  But I have had some anothers learnings like How Spring uses annotations like @Component and @Autowired and how ORM frameworks work
  
 ## 📌 Overview
  This project I show how can be combined custom annotations to dynamically object metadata and simulate database operations like insert records
  
## Key Concepts Used
-I have used notions like:
  -**Custom Annotations**: @interface
  -**Reflections API**: Read annotations and field values
  -**Generics**: <K,V>
